# Managing-personal-schedules

## v1.0.0-beta.1

데스크톱 일정 위젯 – 기능 소개

본 프로그램은 컴퓨터 실행 시 즉시 확인할 수 있는 경량형 데스크톱 일정 관리 위젯입니다.
복잡한 일정 관리 도구를 대체하기보다는, 오늘과 가까운 일정 확인에 최적화된 인터페이스를 제공합니다.

주요 기능

월간 캘린더 제공
직관적인 월 단위 캘린더를 통해 날짜를 선택하고 일정을 관리할 수 있습니다.

일정 추가 · 수정 · 삭제
선택한 날짜에 대해 여러 개의 일정을 등록할 수 있으며, 등록된 일정은 수정 및 삭제가 가능합니다.

오늘 날짜 자동 표시
프로그램 실행 시 오늘 날짜가 자동으로 선택되고 시각적으로 강조되어, 오늘 일정을 즉시 확인할 수 있습니다.

항상 위에 표시되는 위젯 형태
다른 프로그램 위에 항상 표시되며, 바탕화면 위젯처럼 사용할 수 있습니다.

자유로운 위치 이동
마우스로 드래그하여 원하는 위치로 쉽게 이동할 수 있습니다.

우클릭 컨텍스트 메뉴
우클릭을 통해 위젯 숨기기 또는 프로그램 종료가 가능합니다.

단축키를 통한 다시 표시
숨긴 위젯은 단축키로 즉시 다시 표시할 수 있습니다.

투명도 조절 기능
슬라이더를 통해 위젯 투명도를 실시간으로 조절하여 작업 환경에 맞게 사용할 수 있습니다.

미니멀 디자인
불필요한 UI 요소를 제거한 간결한 디자인으로, 장시간 사용 시에도 시각적 부담을 최소화했습니다.

데이터 관리

로컬 저장 방식
모든 일정 데이터는 로컬 SQLite 데이터베이스에 저장되며, 외부 서버나 인터넷 연결이 필요하지 않습니다.
